## Repository Overview

This repository contains my work for Homework 1 in the coding assistant onboarding workflow. It demonstrates a simple but complete process: setting up a Git repository, using a coding assistant to create a draft, manually revising that draft, and documenting the work through commits.

## Repository Structure

```text
Genai_Homework1/
|- README.md
|- assistant_draft.md
|- final_version.md
```

## What Each File Does

| File | Purpose |
| --- | --- |
| `README.md` | Documents the project, Git workflow, commit history, and final video link |
| `assistant_draft.md` | The assistant-generated first draft |
| `final_version.md` | The manually revised version based on the draft |

## Git Steps Used

1. Created or opened the local repository folder.
2. Initialized and checked the Git repository state.
3. Added the required assignment files.
4. Generated the first draft with a coding assistant.
5. Prepared a second file for manual revision.
6. Committed progress incrementally instead of saving everything for one final commit.
7. Planned to push the completed repository to GitHub after the final manual edit and video link update.

## Commit Summary

The repository is being built through small, incremental commits. The working commit plan is:

1. `Initial commit`
2. `Add assistant-generated draft`
3. `Add final version template and README polish`
4. `Manually revise final_version.md`
